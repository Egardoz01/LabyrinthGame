Данная программа представляет из себя классическую игру по прохождению лабиринта.
На начальном экране пользователь видит приветствие и Главное меню. 
Главное меню содержит 2 пункта - Игра и Помощь. 
В пункте Помощь есть подпункт Правила игры. Нажатие на него вызывает форму с правилами игры.
В пункте Игра есть подпункты:
Новая игра
загрузить сохранение
сохранить игру
завершить игру
выход

Пункт Новая игра вызывает форму с выбором уровня сложности, на данный момент есть 3 уровня:
EASY - лабиринт 5 на 5 клеток
MEDIUM - лабиринт 20 на 20
HARD - лабиринт 20 на 40

Чтобы начать игру, нужно выбрать уровень сложности и нажать на кнопку "Начать игру".
После начала  будет сгенерирован лабиринт соответствующего размера. В разных концах лабиринта появятся МЫШ и СЫР. Пользователь может управлять МЫШом с помошью стрелок на клавиатуре.
Чтобы выйграть пользователь должен провести МЫШ к СЫРу. 
Лабиринт сгенерирован специальным алгоритмом. Каждую игру генерируется случайный идеальный лабиринт, так что путь от МЫШа к сыру всегда существует и при этом он только один. 
После прохождения лабиринта открывается форма с поздравленийми и игра снова переключается на начальный экран.

Вов ремя игры можно сохранить игру, для этого нужно будет выбрать папку, куда сохранить файл и имя файла.
Загрузить сохраненную игру можно только с начального экрана. Для этого нужно нажать на соотвествующий пункт меню и выбрать соответствующий файл.
